# ONBOARDING.md
## Welcome to CmdCalc
CmdCalc is an open-source project launch system that provides a command-line calculator with basic arithmetic operations, trigonometric functions, exponential and logarithmic functions, and support for variables and expressions. We're excited to have you on board as a contributor! This guide will help you get started with the project.

### Quick Start Guide
To get a feel for the project, follow these steps:
1. **Explore the repository**: Browse the [CmdCalc repository](https://github.com/your-username/cmdcalc) to familiarize yourself with the codebase.
2. **Run the application**: Clone the repository and run the application using the instructions in the [README.md](README.md) file.
3. **Play with the calculator**: Try out different calculations and features to understand the project's functionality.

### Development Setup
To start contributing to CmdCalc, you'll need to set up your development environment:
#### Prerequisites
* Python 3.8 or later
* pip (the package installer for Python)
* A code editor or IDE of your choice

#### Setting up the Project
1. **Clone the repository**: Run `git clone https://github.com/your-username/cmdcalc.git` to clone the repository to your local machine.
2. **Create a virtual environment**: Run `python -m venv venv` to create a virtual environment.
3. **Activate the virtual environment**: Run `source venv/bin/activate` (on Linux/Mac) or `venv\Scripts\activate` (on Windows) to activate the virtual environment.
4. **Install dependencies**: Run `pip install -r requirements.txt` to install the required dependencies.
5. **Run the tests**: Run `python -m unittest discover` to ensure all tests pass.

### First Contribution Steps
Ready to make your first contribution? Follow these steps:
1. **Find an issue**: Browse the [issue tracker](https://github.com/your-username/cmdcalc/issues) to find a bug or feature that interests you.
2. **Assign the issue**: Comment on the issue to let others know you're working on it, and assign it to yourself.
3. **Create a branch**: Run `git checkout -b your-branch-name` to create a new branch for your changes.
4. **Make your changes**: Edit the code, add tests, and ensure the application still works as expected.
5. **Commit your changes**: Run `git add .` and `git commit -m "your-commit-message"` to commit your changes.
6. **Open a pull request**: Run `git push origin your-branch-name` and open a pull request against the `main` branch.

### Getting Help
Need help or have questions? Don't hesitate to reach out:
* **Issue tracker**: Open an issue on the [issue tracker](https://github.com/your-username/cmdcalc/issues) for help with a specific problem.
* **Discussion forum**: Join the [discussion forum](https://github.com/your-username/cmdcalc/discussions) for general questions and conversations about the project.
* **Contact the maintainers**: Reach out to the project maintainers directly via email or private message for sensitive or urgent matters.

Remember to always follow the project's [code of conduct](CODE_OF_CONDUCT.md) and [contributing guidelines](CONTRIBUTING.md) when interacting with the community and contributing to the project. Happy coding!
