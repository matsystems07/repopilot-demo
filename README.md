# CmdCalc
================

## Overview
CmdCalc is a simple and efficient command line calculator designed to perform mathematical calculations with ease. It provides a wide range of functions, including basic arithmetic operations, trigonometric functions, exponential and logarithmic functions, and support for variables and expressions.

## Problem Statement
Users often need to perform mathematical calculations from the command line, but existing solutions can be cumbersome and limited in their functionality. CmdCalc aims to fill this gap by providing a simple, intuitive, and fully functional command line calculator.

## Key Features
* Basic arithmetic operations: addition, subtraction, multiplication, and division
* Trigonometric functions: sine, cosine, tangent, and their inverses
* Exponential and logarithmic functions: exponential, natural logarithm, and base-10 logarithm
* Support for variables and expressions: store and reuse values, and perform calculations using expressions

## Intended Users
CmdCalc is designed for anyone who needs to perform mathematical calculations from the command line, including:
* Students: for homework, projects, and research
* Developers: for quick calculations and testing
* Mathematicians: for exploratory calculations and prototyping

## Setup Instructions
To get started with CmdCalc, follow these steps:
1. Clone the repository: `git clone https://github.com/your-username/CmdCalc.git`
2. Navigate to the project directory: `cd CmdCalc`
3. Install dependencies: `pip install -r requirements.txt` (if applicable)
4. Run the calculator: `python cmdcalc.py`

## Usage
To use CmdCalc, simply run the calculator and start entering commands. The calculator supports the following syntax:
* Basic arithmetic operations: `2 + 2`, `3 * 4`, etc.
* Trigonometric functions: `sin(3.14)`, `cos(0)`, etc.
* Exponential and logarithmic functions: `exp(2)`, `log(10)`, etc.
* Variables and expressions: `x = 5`, `y = x * 2`, etc.

Type `help` to view a list of available commands and functions.
